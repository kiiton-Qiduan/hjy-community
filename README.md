# 合家云社区物业管理系统（Hjy Community）

## 项目简介

合家云社区物业管理系统是一个基于 Spring Boot 的前后端分离项目，旨在模拟真实物业管理业务场景，实现社区资产管理、业主管理、权限管理、系统监控等核心功能。

本项目作为个人学习项目，主要用于深入学习企业级 Java 后端开发技术，包括 Spring Boot、MyBatis Plus、Redis、Spring Security、JWT 等主流技术栈，并积累实际项目开发经验。
